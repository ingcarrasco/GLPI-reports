# GLPI-Reports

Es un proyecto para analizar los tickets creados en GLPI 9.5.0

## Ejecutar dashboard
```sh
git clone https://github.com/ingcarrasco/GLPI-reports.git && cd ./GLPI-reports && docker compose up -d
```
